Project description :
Technical Test realize for Iggy Paris. The goal of the technical test is to realiz a quest system with an UI.

Commands :

    On keyboard and mouse:
        W/S : Move forward/backward
        A/D : Move left/right
        Mouse : Look around 
        Left Click : Aim
        F : To Interact
